# aqua.nvim

`aqua.nvim` is the Neovim colorscheme for the broader **Aqua** theme ecosystem.

This repo contains the Neovim theme itself. It is intended to pair with the Omarchy theme repo:

- **Omarchy theme repo:** `omarchy-aqua-theme`
- **Neovim theme repo:** `aqua.nvim`

## What this repo is

This repo is focused on Neovim only.

It provides the editor colorscheme, highlight groups, and plugin styling needed for Neovim to match the rest of the Aqua theme.

## Relationship to Omarchy

`aqua.nvim` is the editor-side counterpart to `omarchy-aqua-theme`.

That means:

- `aqua.nvim` handles Neovim colors and highlights
- `omarchy-aqua-theme` handles desktop and app theming for Omarchy
- both are meant to share the same palette and visual direction

## Theme pairing

Use these together for a matching setup:

- **Neovim:** `aqua.nvim`
- **Omarchy:** `omarchy-aqua-theme`

## Goal

The goal of `aqua.nvim` is to make Neovim feel like a natural part of the wider Aqua environment instead of a separate, mismatched editor theme.
